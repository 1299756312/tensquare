# tensquare
这是一个springcloud集成 mysql、redis、mongoDB 、mq、ES等中间件的项目案例    
使用这个项目，你可以快速上手springboot 各个微服务的搭建、以及springcloud各个组件之间的交互
下载时，记得修改各个服务的端口号
# 服务版本
jdk 1.8  
springboot 2.0.1.RELEASE  
mysql 5.7  
redis 5.x  
mongoDB latest  
rabbitmq latest  
es 5.6.8  
jenkins latest  
docker 18.03.1-ce
